The Humboldt Helpers

Built at Cal Poly DXHub AI Summer Camp, this solution reduced website clickrage for Humboldt University researchers by delivering faster, contextual answers through RAG on AWS.

Tech Stack: Streamlit, CSS, AWS Bedrock, AWS S3, Amazon Knowledge Base, RAG, Semantic Chunking, Titan V2 Embeddings, Claude Sonnet, NLP, Prompt Engineering, Guardrails, GitHub.

During the Cal Poly DXHub AI Summer Camp, my team and I developed a solution to help Humboldt University’s research department navigate their website more efficiently, addressing clickrage caused by multiple links on a single page.

We designed a responsive interface using Streamlit and CSS with modern UI/UX inspired by Humboldt branding, then implemented Retrieval Augmented Generation (RAG) using Amazon Bedrock RetrieveAndGenerate.

  • Used Amazon S3 Buckets, S3 vector database, and Amazon Knowledge Base for retrieval quality and scale.
  • Applied Semantic Chunking, Titan V2 Embeddings, and Claude Sonnet for stronger contextual responses.
  • Implemented Guardrails and prompt controls to support safe, accurate, and reliable AI behavior.
  • Documented progress in GitHub and collaborated with students across the CSU system.

I am grateful to Ryan Matteson, Nick Osterbur, Darren Kraker, and the admin team for creating a transformative camp experience.
